# ETFTrueHoldings

**An open source ETF portfolio X-ray tool.**

ETFTrueHoldings helps investors understand what they actually own. By aggregating the holdings of multiple ETFs in your portfolio, it calculates your true exposure to individual companies and sectors.

## Features

*   **Portfolio Aggregation**: Add up to 50 ETFs and assign equity values to see the combined weight of underlying assets.
*   **True Exposure**: Discover how much of your money is actually invested in specific companies (e.g., NVDA, AAPL) across overlapping ETFs.
*   **Visualization**: Interactive charts for top underlying assets and sector allocation.
*   **Detailed Breakdown**: Searchable, paginated list of all underlying holdings with asset class information.
*   **Smart Caching**: Local caching to optimize AlphaVantage API usage.

## Setup

1.  Get a free API key from [AlphaVantage](https://www.alphavantage.co/support/#api-key).
2.  Enter the key in the app settings.
3.  Start building your portfolio.
